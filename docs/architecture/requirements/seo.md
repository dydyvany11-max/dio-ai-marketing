# Функциональные требования

## Анализ Core Web Vitals

 - Входные параметры: URL адрес сайта, который нужно проанализировать
 - Система использует Google PSI API для технического SEO
 - Возвращается в виде деатльного ответа, который понятен пользователю не погруженному в SEO (AI синтезирует данные в понятный ответ)
 - Результат анализа можно визуализировать в любом удобном для пользователя формате (различные диаграммы, инфографика, схемы, mermaid)

## Семантичекий анализ страницы

 - Входные параметры: URL адрес страницы
 - Извлечение семантического ядра
 - Кластеризация ключевых слов и тем
 - Оптимизация HTML разметки, тэгов и структуры страницы
 - Генерация оптимальных JSON-LD скриптов
 - Оптимизация страницы под AIO
 - Генерация контента на основе базы знаний
 - Возможность загрузить базу знаний (поддежка документов: pdf, docx, pptx, xlsx, png, ...)
 - Просмотр качества оптимизации страницы в виде различных диаграмм
 - Обзор текущих ошибок и рекомендаций по их улучшению + насколько процентов улучшится оптимизация страницы, если исправить ошибку

## Технический аудит сайта

 - Ввод: URL сайта, глубина сканирования (по умолчанию 3)
 - Проверка и помощь в составлении sitemap.xml, robots.txt, llms.txt
 - Анализ перелинковки страницы
 - Отчёт содержит: список ошибок с указанием страниц, рекомендации по исправлению
 - Ошибки группируются по типу (4xx, 5xx, meta issues и др)
 - Возможность повторного сканирования после исправлений

## Анализ UI/UX

 - Интеграция с сервисом "Яндекс Метрика", используя OAuth (пользователь даёт согласие на получение данных из его личного кабинета)
 - Генерация и проверка сценариев тестирования UI основываясь на UX
 - Построение воронок
 - Анализ скриншотов интерфейса (мобильные телефоны, ПК, ...)
