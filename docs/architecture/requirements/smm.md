# Функциональные требования

**Поддерживаемые соцсети:** ВК, Telegram

## Анализ аудитории

 - Входные параметры: ID группы, канала или ссылка на него
 - Кластеризация аудитории по интересам, возврасту, активности и.т.д
 - Выявление конкурентов
 - Создание портрета пользователя
 - Удобные дашборды для просмотра отчётов

## Генерация контента

 - Входные параметры: тема, промпт (запрос пользователя), тон
 - Использование отчёта по аудитории для более точной генерации
 - Возможность загрузить базу знаний компании (шаблоны, правила написания постов, идеи для контента и.т.д) для более точной генерации
 - Генерация различных типов контента: текст, изображения, сценарии для видео, ...
 - Оптимизация постов под конкретную платформу
 - Интеграция с API соцсетей для возможности публикации поста из личного кабинета

## Сводный отчёт по аккаунтам

 - Отчёт включает: суммарный охват, вовлечённость (ERR), прирост подписчиков, лучшие посты
 - Экспорт данных в PDF, DOCX
 - Возможность настроить отчёт: выбор периода, метрик, соцсеть
 - Автоматическая отправка отчёта на email по расписанию

## Мониторинг и анализ трендов

 - Система должна анализировать тренды, инфоповоды на различных платформах
 - Поиск и анализ упоминаний компании
